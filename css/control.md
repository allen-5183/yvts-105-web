# 使用 CSS 控制頁面

1. 嵌入式: 直接寫在標籤 &lt;h2&gt; 的屬性="值" 上.  
   ``` 
   <h2 style="color:red;"> Hello </h2>
   
   ```   
2. 內部: &lt;style&gt; 標籤, 寫在網頁內容裏. 
   ```
          <style type="text/css">
          h2{                                # selector {
             color: red;                     #              屬性: 值
          }                                  # }
        </style>

   ```
